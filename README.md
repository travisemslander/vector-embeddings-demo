# vector-embeddings-demo
Demonstration of vector embeddings and cosine distances 

This is a simple tailwind css static web app that uses web assembly to run
Xenova/all-MiniLM-L6-v2 embedding model from hugging face.

This demonstrates how embeddings can be generated from text and words or sentences
with similar concepts produce vectors that are closer together based on cosine distance.

This simple app was generated using Gemini 3.1 Pro
